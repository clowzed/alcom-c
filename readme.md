<div align = "center">
<h1>Installation</h1>
</div>

```bash
git clone https://github.com/clowzed/alcom-c.git
cd alcom-c
make install
```

<div align = "center">
<h1>Usage</h1>
</div>

```bash
alcom file1.asm file2.asm file2.asm
```
